Tutos : 
- tuto général : **!!tuto**
- tuto pour les combattants (Bully) : **!!tuto_bully**
- tuto des combats : **!!tuto_fight**
- tuto qui explique les raretés : **!!tuto_rarity**
- tuto sur l'achat des combattants (Bullies) : **!!tuto_shop**
- tuto pour les donjons : **!!tuto_dungeon**
- tuto pour les ruines : **!!tuto_ruin**
- tuto sur les divers consommables : **!!tuto_conso**
- tuto qui affiche les effets de tous les buffs de combats : **!!list_buffs**

Pour voir toutes les commandes : **!!help**
